CSS と JavaScript と HTML から 空白 や 改行, コメント など余分な要素を削除して, コードを軽量化（圧縮）します.
ファイル種別は URI の拡張子で判定します. （CSS: .css, JavaScript: .js, XML: .xml, HTML: その他全て）

ファイル生成機能について

ファイル生成は 標準ページ のみ対象です.
ファイル生成をONにすると, ページ更新時に CSS／JavaScript ページの 実ファイル を生成します.
ファイルは URI（ページのアドレス）に対応するディレクトリに生成し, 以後, 実ファイルが優先して読み込まれます.
* 対応するディレクトリが存在しない場合には ファイルを生成しません. 事前に FTP 等で作成してください.
