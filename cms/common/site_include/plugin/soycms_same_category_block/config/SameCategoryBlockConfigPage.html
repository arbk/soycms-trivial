プラグインブロックで表示している記事のカテゴリーと同じカテゴリーの記事一覧を表示します。
※ブログの記事毎ページのみ記事一覧を取得できます。

設定方法

テンプレートのブロック設定でプラグインブロックを選び、プラグインのセレクトボックスでsame_category_blockを選びます。

タグの記述例

他のブロックの時と同じですが、block:id="***"の記述の時に、表示件数を指定することができます

<!-- block:id="***" cms:count="10" cms:labels="2,3" cms:random="1" -->
表示したい内容のcms:idを追加
<!-- /block:id="***" -->

***の箇所は任意の値です。
cms:countで検索結果を何件表示するか決めることができます。
※cms:countの指定がなくても動作します。

cms:labelsはブログページで設定したカテゴリ分けで使用するラベルの内、表示したいラベルのIDをカンマ区切りで指定できます。
ラベルIDは管理画面でラベルの詳細を開き、ブラウザのアドレスバーに入力されているURLの末尾の数字になります。
例："http://example/cms/soycms/index.php/Label/Detail/2"の場合のラベルIDは2になります。

cms:randomは1を指定すると表示順がランダムになります。
