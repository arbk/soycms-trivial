使い方

テンプレートをファイルに書き出すには

現在使われているページテンプレートを実ファイルに書き出すには、ファイルに書き出すボタンをクリックしてください。書き出しディレクトリにファイルとして出力されます。
書き出されたファイル名は通常ページは「ページID_ページのアドレス」で、ブログページは「ページID_ページのアドレス_テンプレートの種類」となります。 「URLの書き換えを行う」チェックボックスにチェックを入れると、 
<img src="サイトID/..." />
などの相対URLが 
<img src="http://ドメイン/サイトID/..." />
の様に絶対URLに書き換えられて出力されます。この書き換えはhref, srcの属性値に適用されます。

書き出したファイルをSOY CMSのテンプレートにするには

書き出した後に変更のあったファイルがファイルから読み込むボタンの上に表示されます。読み込みたくないファイルのチェックをはずすことで読み込まれなくなります。
書き出したテンプレートファイルを再びSOY CMSのテンプレートにするためには、ファイルから読み込むボタンをクリックしてください。
同期を取るファイルのファイル名は、書き出したファイル名と同一である必要があります。
URLの書き換えを行うチェックボックスにチェックを入れると、テンプレートを書き出した際のURLの書き換えと逆の変換を行います。
