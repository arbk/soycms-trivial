設定

【置換文字列の指定】
対象文字列と置換後文字列
【書式】 "対象文字列":"置換後文字列", 1行1組
【置換対象の指定】
置換対象を指定してください.
  • 本文（content）
  • 追記（more）
【ラベルとの関連付け】

文字列の置換処理を特定のラベルと関連付ける場合は ラベルID（数字）を指定してください.
指定されたラベルが付いた記事でのみ置換処理を実行します. （空の場合は全ての記事で置換処理を実行します.）

使い方

対象文字列と置換後文字列は ""ダブルクォーテーションで囲み, ":"コロンで組にしてください.
対象文字列に空白を指定することはできません. 置換後文字列に空白を指定することは可能です.
文字列に改行を含めることはできません.
複数指定する場合は改行してください.

  • 記事更新時に文字列が置換されます. ※記事データ自体が更新されます. 注意してください.
  • 正規表現やワイルドカード(*)は使えません.
