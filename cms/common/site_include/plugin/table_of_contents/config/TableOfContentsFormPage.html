プラグインの概要

記事投稿画面で下記のように見出しタグ入りに記事を作成すると、

<h1>H1の文字列</h1>
<h2>H2の文字列1</h2>
<h3>H3の文字列1</h3>
<p>ここから本文</p>
<h3>H3の文字列2</h3>
<p>ここから本文</p>
<h2>H2の文字列2</h2>
<h3>H3の文字列3</h3>
<p>ここから本文</p>
<h2>H2の文字列3</h2>
<h3>H3の文字列4</h3>
<p>ここから本文</p>

公開側のブログページの記事詳細ページで

<ul>
	<li>
		<a href="#heading-1-1" class="layer-1">H1の文字列</a>
		<ul>
			<li>
				<a href="#heading-2-2" class="layer-2">H2の文字列1</a>
				<ul>
					<li><a href="#heading-3-3" class="layer-3">H3の文字列1</a></li>
					<li><a href="#heading-3-4" class="layer-3">H3の文字列2</a></li>
				</ul>
			</li>
			<li>
				<a href="#heading-2-5" class="layer-2">H2の文字列2</a>
				<ul>
					<li><a href="#heading-3-6" class="layer-3">H3の文字列3</a></li>
				</ul>
			</li>
			<li>
				<a href="#heading-2-7" class="layer-2">H2の文字列3</a>
				<ul>
					<li><a href="#heading-3-8" class="layer-3">H3の文字列4</a></li>
				</ul>
			</li>
		</ul>
	</li>
</ul>

記事の構成に従って、任意の箇所に見出しを出力できます。

使い方

記事投稿画面、もしくはブログページの記事詳細画面で##HEADING##を入れておくと、ページの表示時に##HEADING##が置換されて見出しが表示されます。

※注意

記事投稿時に<h2>の下に<h4>と見出しの番号が連番でないと見出しの自動作成時に<h4>は解釈されないので、見出しタグは連番で使用してください。
