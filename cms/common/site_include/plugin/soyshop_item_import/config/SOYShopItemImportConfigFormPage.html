連携するSOY Shopのサイトを選択してください。

 

SOY CMSのブログ記事でSOY Shopにある商品を取り上げることができます。

記事投稿画面を開くと、商品コードを入力するフォームが表示されています。
フォームに商品コードを入力して記事を投稿すると、
SOY Shopのデータベースから商品情報を取得し、
その情報を公開しているブログに表示することができるようになります。


記事の表示に追加されるcms:id

cms:id タグ 説明
item_id 全て 商品IDを出力します
item_code 全て 商品コードを出力します
item_name 全て 商品名を出力します
item_alias 全て 商品のエイリアスを出力します
item_link A 商品詳細で入力したリンクを出力します
item_small_image img 商品のサムネイル画像(小)を出力します
item_large_image img 商品のサムネイル画像(大)を出力します
item_price 全て 商品の価格を出力します
item_sale_price 全て 商品のセール価格を出力します
※カスタムフィールド以外の商品用のcms:idはすべて使えます
